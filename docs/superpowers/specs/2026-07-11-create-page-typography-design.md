# 创建页字体与长内容设计

## 目标

根据页面标注调整 AI 创建页的字号、简历正文高度和上传区间距，保持现有交互与数据逻辑不变。

## 设计

- Header 主导航、用户胶囊和右侧按钮字号统一下调，Logo 保持现状。
- 简历正文展示字号由 11px 调整为 13px，保持舒适行高。
- 移除正文卡片的固定最小高度，短内容随内容自然收缩。
- 正文超过 360px 时在卡片内部滚动，避免长项目经历拉高整个向导。
- 缩小段落间距，保留解析后的分行语义，不自动合并文本。
- 增大上传图标与标题、说明之间的纵向间距。

## 验证

- 前端源码测试锁定关键 CSS 数值。
- 运行 `npm test`、`npm run build` 和 `git diff --check`。
- 不自动执行简历上传；上传后的视觉效果由用户手工验证。
