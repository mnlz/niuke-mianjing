# 当前分析配置页静态原型设计

## 目标

根据 `http://127.0.0.1:3000/ai-analysis/create` 当前浏览器状态生成一个可独立打开的单文件 HTML，用于离线评审和后续界面迭代。

## 数据范围

- 当前步骤：第 5 步“个人简历”。
- 当前公司与岗位预览：腾讯、后端开发、校招、8 篇面经。
- 当前简历：刘子扬，PDF 1 页，解析 2,145 字，包含教育经历、实习经历、项目经历、专业技能和个人总结。
- 保留当前页面中的姓名、手机号和邮箱；文件仅放入本地原型目录，不进入前端 `public/`。

## 页面与交互

- 复刻顶部导航、七步配置导航、主编辑区和右侧实时报告预览。
- 基本信息使用可编辑输入框。
- 五个简历分区默认以阅读卡片展示，可切换为编辑状态。
- 高级编辑使用折叠区展示完整原始文本。
- 确认按钮、步骤按钮和报告预览状态提供纯前端演示反馈。

## 技术与验证

- 单个 HTML 内嵌 CSS、数据和 JavaScript，不请求后端或 CDN。
- 使用语义化 HTML 和原生控件，支持窄屏响应式布局。
- 合同测试验证当前姓名、联系方式、五个分区、编辑交互脚本和无外部资源依赖。
