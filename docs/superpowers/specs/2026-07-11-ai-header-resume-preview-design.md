# AI Header 与简历预览设计

## 目标

将 AI 分析区域 Header 调整为参考图的横向产品导航，同时保留 OfferLens Logo；为已解析的本地 PDF 简历提供站内弹窗预览。

## Header

- 左侧保留 OfferLens Logo，并继续作为返回首页入口。
- 主导航依次为「AI 分析」「新建分析」「报告示例」「我的报告」。
- 当前页面使用浅蓝背景和蓝色文字标识。
- 右侧显示圆形中文头像与用户显示名组成的胶囊；点击后提供退出登录操作。
- 最右显示蓝色「我的报告」按钮，沿用登录检查逻辑。
- 移动端继续使用紧凑图标导航，不影响现有登录能力。

## 简历预览

- 简历文件栏同时提供「预览」和「重新上传」按钮。
- 解析成功后通过 `URL.createObjectURL(file)` 保存当前会话的本地 PDF 地址。
- 点击「预览」后在 Ant Design Modal 中用原生 `iframe` 展示 PDF。
- 替换简历或组件卸载时调用 `URL.revokeObjectURL` 释放旧地址。
- 没有本地 PDF 地址时不展示可用的预览按钮；粘贴文本不伪装成 PDF。

## 验证

- 源码回归测试确认四个导航项、用户胶囊、报告按钮和 PDF 弹窗存在。
- `npm test` 与 `npm run build` 必须通过。
- 浏览器检查 Header 布局和简历预览按钮；文件选择仍由用户手动完成。
