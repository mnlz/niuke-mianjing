# AI 分析创建页横版布局设计

## 目标

参考用户提供的界面，将 AI 分析创建页从三栏向导改为横向步骤导航与全宽内容面板，重点优化简历确认步骤的信息密度和编辑体验。

## 全局布局

- 七步导航移动到页面标题下方，采用横向连接式步骤条；完成、当前和未开始状态分别使用绿色、蓝色和灰色。
- 删除右侧实时预览栏，主向导面板占满页面内容宽度。
- 所有七个步骤继续复用同一导航和面板结构，操作逻辑保持不变。
- 窄屏下步骤条横向滚动，面板保持单列。

## 简历步骤

- 标题调整为“确认你的简历内容”，展示解析完成说明和草稿保存状态。
- 简历结果使用大卡片：解析状态与页数、字数、分区数置于头部。
- 上传控件改为文件信息横条，显示文件名、PDF 类型和重新上传提示。
- 基本信息使用头像、姓名、邮箱和手机号横向输入布局。
- 五个简历分区使用横向标签切换，每次只展示一个分区，避免长页面同时展开所有内容。
- 当前分区保留阅读/编辑切换，高级原文编辑和确认逻辑保持不变。

## 验证

- 源码合同测试验证横向步骤条、无实时预览、文件信息栏和分区标签。
- 运行前端测试与生产构建。
- 在当前浏览器数据下验证步骤切换、分区切换和编辑状态。
